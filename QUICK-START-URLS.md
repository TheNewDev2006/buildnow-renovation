## URL Parameter Breakdown

### Required Parameters
1. **bn** (Business Name) - Auto-generates logo initials
   - "Acme Corporation" → Initials "AC"
   - Must be URL-encoded (spaces as +)

### Recommended Parameters
2. **e** (Email) - Shown in footer
   - Must be valid email format
   - Auto-sanitized

3. **p** (Phone) - Shown in header
   - Format: +1 (555) 123-4567
   - Auto-validates

4. **c** (Brand Color) - Logo circle color
   - Hex format only: #RRGGBB
   - Must be URL-encoded (%23 instead of #)

### Optional Parameters
5. **a** (Address) - Shown in contact section
6. **w** (Website) - Fallback displayed
7. **fb, tw, li, ig** - Social media links

---

## URL Encoding Reference

| What | How |
|------|-----|
| Space | Use `+` not `%20` |
| # symbol | Use `%23` |
| & symbol | Use `%26` if needed |
| / symbol | Use `%2F` if in data |

### Example Wrong → Right
```
WRONG: ?bn=Acme Corp&c=#F45B1E
RIGHT: ?bn=Acme+Corp&c=%23F45B1E
```

## Troubleshooting URLs

### Logo Not Updating?
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
- Clear browser cache
- Check URL is exactly right

### Color Not Showing?
- Ensure color starts with %23 not #
- Valid format: %23 followed by 6 hex digits
- Example: %235B7FE0 for #5B7FE0

### Text Broken or Showing Strangely?
- Check spaces are encoded as + not spaces
- Avoid special characters in company name
- Use standard ASCII characters

### Link Not Loading?
- Verify localhost:8000 is running (`php -S localhost:8000`)
- Check spelling of index.php or index-7.php
- Try without parameters first: `?bn=Test`

## URL Generator (Copy-Paste Formula)

```
http://localhost:8000/index.php?bn=COMPANY_NAME&e=PROSPECT_EMAIL&p=PROSPECT_PHONE&a=COMPANY_ADDRESS&c=%23COLOR_HEX
```

Replace:
- `COMPANY_NAME` → with spaces as `+`
- `PROSPECT_EMAIL` → valid email
- `PROSPECT_PHONE` → format: +1 (555) 123-4567
- `COMPANY_ADDRESS` → main office location
- `COLOR_HEX` → 6-digit hex code (e.g., F45B1E)

Example filled in:
```
http://localhost:8000/index.php?bn=ABC+Corp&e=hello@abccorp.com&p=+15551234567&a=123+Elm+St&c=%23F45B1E
```

If a prospect finds the template URL without parameters, they'll see:
- **Logo**: CI (Cynix Inc)
